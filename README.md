# 🌌 Acerca del proyecto

**Flocial** es un protocolo social descentralizado desarrollado para el ecosistema Flow que permite gestionar identidades sociales Web3 y crear conexiones entre usuarios, aplicaciones y comunidades blockchain.

La plataforma permite:

- 👤 Crear perfiles descentralizados
- 🌐 Gestionar identidades Web3
- 🔎 Descubrir usuarios
- 🤝 Facilitar networking
- ⚡ Integrar APIs rápidas
- 📡 Compartir datos sociales
- 🔗 Conectar dApps del ecosistema Flow

La plataforma fue desarrollada con enfoque en:

- 🌐 Descentralización
- ⚡ Escalabilidad
- 🔒 Seguridad blockchain
- 🧩 Integración modular
- 📡 APIs de baja latencia
- 🚀 Ecosistemas Web3
